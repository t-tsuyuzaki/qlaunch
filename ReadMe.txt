qlaunchは，DOSかつパラレル時代のaltera quartus programmerをWindowsからGUIで，かつ量産目的のボタン1個でサクサク利用するためのランチャーです．

2005年にとある企業から依頼されて作成しましたが，その依頼はうやむやのまま無かった事になりその後どうなったのかも判りません．
契約書を交わしてから作れば良いものを，当時若造だった自分はそんな事をせず個人信用だけで進めてたのがすべての要因ですね．

それはともかく当時の自分としてはなかなかの習作（DOSプログラムを呼び出したり実行結果を読み取って反応したり）でしたので，
githubを本格的に使い始める意味でも今更誰も使わない意味の無いものですがMITライセンスで公開してみるテストです．
